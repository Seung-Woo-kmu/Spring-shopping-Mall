# Spring-shopping-Mall
### 신발 쇼핑몰 프로젝트
1. 개발 인원 - 1명

2. 개발 기간 - 약 1달

3. 기술 스택
- Spring Boot, Spring Data JPA, JPA(Java Persistent API), MariaDB, Java, Javascript, HTML, CSS, Bootstrap

4. 주요 기능
- 일반회원:  로그인, 회원가입, 아이디 비밀번호 찾기, 장바구니, 상품 주문, 상품 리뷰
- 관리자: 관리자 페이지, 상품 추가 삭제 및 수정, 리뷰 삭제, 주문 취소
